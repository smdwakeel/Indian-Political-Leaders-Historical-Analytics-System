 Indian Leaders Analytics Dashboard

 Project Overview

The Indian Leaders Analytics Dashboard is a comprehensive data analytics project developed to analyze and visualize information about important constitutional and political leaders of India. The project demonstrates the complete data analytics workflow starting from database design and management, data extraction and processing, and ending with interactive business intelligence reporting.

This project was built using MySQL, Python, and Power BI to showcase how raw data can be transformed into meaningful insights through modern analytics tools.

---

 Objectives

The primary objectives of this project are:

* Design and manage a structured relational database using MySQL.
* Store and organize information related to Indian constitutional leaders.
* Extract and process data using Python and Pandas.
* Generate analytical datasets for reporting purposes.
* Build an interactive Power BI dashboard for visualization and decision-making.
* Demonstrate an end-to-end Data Analytics workflow.

 Database Design

The database was designed using a relational structure and contains information about various leadership positions in India.

Tables included:

* President
* Vice President
* Prime Minister
* Governor
* Chief Minister
* Deputy Chief Minister
* Speaker of Lok Sabha
* Chief Justice of India

The database schema was carefully designed to maintain consistency, scalability, and efficient querying.

---


 Data Processing with Python

Python was used to establish a connection with the MySQL database and automate data extraction.

Major tasks performed:

* Connected Python with MySQL database.
* Retrieved records from multiple tables.
* Combined data into a unified dataset.
* Performed preliminary data validation.
* Exported the final dataset into CSV format.
* Prepared the data for Power BI reporting.

This step demonstrates practical ETL (Extract, Transform, Load) operations commonly used in real-world analytics projects.
